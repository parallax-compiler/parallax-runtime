# Parallax Runtime

Hardware-agnostic GPU runtime providing unified memory and kernel execution for C++ parallel algorithms.

## Quick Start

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
sudo make install
```
